Blocks

by using template, we can load same html code components between different pages.
and different code component will be loaded by block.
when loading template html in actual page html, blocks will be automatically replaced when showing by view.
it can respect Django's DRY rule.

for ... in

there is a for loop in template language.
it helps put repeating elements in short.
usage: {\% for post in posts %} #what_to_iterate {\% endfor %}

if condition

there is a if / else in template language.
it helps showing element by condition.
usage: {\% if (condition) %} #what_to_do {\% else %} {\% endif %}

showing variables

we can put variables in templates.
it helps easily reusing template and respect DRY.
usage: \{\{variable_name}}
